CurricuForge is a Generative AI-powered system that automatically creates structured university-level curricula based on a given course topic. It helps educators and institutions design professional, industry-aligned course structures quickly and efficiently.

## Features
- Generates complete course structure
- Provides course overview and objectives
- Creates module-wise breakdown
- Suggests learning outcomes
- Built using Streamlit and Hugging Face API

## Tech Stack
- Python
- Streamlit
- Hugging Face Inference API

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your Hugging Face API key in a .env file:
   HF_API_KEY=your_api_key_here
4. Run the app:
   streamlit run app_streamlit.py
